## Descripción

Este programa muestra el uso de diferentes estructuras de datos en Java:

### LinkedList

Se crea una lista enlazada, se agregan nombres y se realizan operaciones como obtener el primer, último y un elemento específico de la lista.

### Queue (Cola)

Se utiliza una cola para almacenar personas y atenderlas en el orden en que ingresan (FIFO: First In, First Out).

### Stack (Pila)

Se utiliza una pila para almacenar elementos y retirarlos en orden inverso al que fueron agregados (LIFO: Last In, First Out).

### Ejercicio 1: Invertir una Cadena

Se crea un método que invierte una palabra usando únicamente una pila.

**Ejemplo:**

```text
COMPUTACION
```

Resultado:

```text
NOICATUPMOC
```
